Evnia - React Documentation

Introduction

Evnia is a React and Bootstrap 4 based Multipurpose templates. This has been built with React.js! Evnia is a creative responsive template purposely designed and built for Multipurpose Event and Conference website.

Setup Development Environment

In order to kickstart the development of the web app with Evnia, the very first thing need to do is to setup a react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • Node JS works as a node package manager behind the scene. It is recommended to download and install the latest version from the Node JS official site http://nodejs.org/.

Preparation

You'll need to install Node.js >=v8.11+ (Recommended Version) (NPM comes along with it) and serve (a NPM package)

Use command line tool i.e. powershell or terminal and navigate to the project(Evnia) root. This is powershell in windows and terminal on mac/linux.

Note:

This project set up with Express.js backend framework, so you have to decide what you do, run with express.js? or without express.js

Let's looking how setup with Express.js & without Express.js:
With Express.js

Just open terminal inside the project root & run: npm install after that move forward one directory such as client cd client and run npm install again that is for client, and again move back to one directory such ass cd .. now run npm run dev and this will return something like the following:

> evnia@1.0.0 dev D:\tf-jsx\evnia-react

> concurrently "npm run server" "npm run client"

> evnia@1.0.0 server D:\tf-jsx\evnia-react

> node index.js

> evnia@1.0.0 client D:\tf-jsx\evnia-react

> npm run start --prefix client

> Read on http://localhost:5000 (It is backend server url with port)

> evnia-react@0.1.0 start D:\tf-jsx\evnia-react\client

> react-scripts start

Compiled successfully!

You can now view evnia in the browser.

Local: http://localhost:3000/

On Your Network: http://192.168.1.6:3000/

Now in the browser go to localhost:3000

Without Express.js

Just pick the client folder and inside that open the terminal and run: npm install and this will import all the dependencies for the Evnia React. After that run npm start and this will return something like the following:

Starting the development server...

Compiled successfully!

Local: http://localhost:3000/

On Your Network: http://192.168.1.6:3000/

Now in the browser go to localhost:3000

Fonts

By default, the template loads Poppins font from Google Web Font Services, you can change the font with the one that suits you best.

You will find the font code in the "style.scss" file: /client/src/assets/css/style.scss


@import url('https://fonts.googleapis.com/css?family=Poppins:100,100i,200,200i,300,300i,400,400i,500,500i,600,600i,700,700i,800,800i,900,900i');
